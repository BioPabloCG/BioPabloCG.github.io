# Pablo Castillo García — Portfolio

Personal academic and professional portfolio, live at [biopablocg.github.io](https://biopablocg.github.io).

## About

A single-page portfolio built from scratch in pure HTML, CSS, and vanilla JavaScript — no frameworks, no templates, no build step.

## Structure

```
BioPabloCG.github.io/
├── index.html        # Main portfolio page (single file — all CSS and JS included)
├── avatar.jpeg       # Profile photo
└── img/
    ├── work-1.png    # Selected Work — Publication (Fermentation, 2024)
    ├── work-2.png    # Selected Work — Personal project (IPM Pest Identifier)
    ├── work-3.png    # Selected Work — Master's Thesis (CABIMER)
    └── work-4.png    # Selected Work — Research in progress
```

## Sections

- **About** — Introduction about myself
- **Research Experience** — timeline with supervisors and tech stack per role
- **Publications** — Papers I have participated in
- **Projects** — personal and open-source work to develop my skills
- **Skills** — grouped by domain: programming, bioinformatics, DevOps, research methods, arts & communication
- **Education** — degrees and additional education
- **Selected Work** — figures and outputs from research, thesis, and personal projects

## Local preview

No build step required. Open `index.html` directly in any browser. An internet connection is needed for Google Fonts to load correctly.

## Tech stack

- HTML5 / CSS3 / Vanilla JavaScript
- [Google Fonts](https://fonts.google.com) — Cormorant Garamond, Source Sans 3, JetBrains Mono
- Deployed via [GitHub Pages](https://pages.github.com)
